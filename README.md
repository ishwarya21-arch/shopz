# ShopZen 🛒

ShopZen is a modern e-commerce web application built using HTML, CSS, and JavaScript. It provides a clean shopping interface with product browsing and cart functionality.

## Features
- 25 products
- Product search
- Add to cart
- Responsive layout
- Modern UI design
- Interactive product cards

## Technologies Used
- HTML5
- CSS3
- JavaScript
